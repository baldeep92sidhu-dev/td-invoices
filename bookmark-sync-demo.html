📱💻 Bookmark Sync Demo

Sync your bookmarks across phone, laptop, and GitHub

🚀 Getting Started (Step-by-Step)

  1. Create GitHub Personal Access Token:
    • Go to github.com/settings/tokens
    • Click "Generate new token (classic)"
    • Select scope: repo
    • Copy the token
  2. Paste your token below and click "Set GitHub Token"
  3. Add bookmarks using the form
  4. Click "Sync to GitHub" to save bookmarks to your repo
  5. On another device: Paste the same token and click "Sync from GitHub"
